Sobre o Bot

Seu Bot de Música é um bot incrível que reproduz músicas diretamente do YouTube. Estamos trabalhando constantemente para implementar recursos adicionais, como suporte para serviços de streaming como o Spotify. Nossa missão é fornecer a melhor experiência musical para os usuários do Discord.

Comandos

?play

Use este comando para tocar uma música do YouTube. Por exemplo: ?play nome da música

?skip

Use este comando para pular a música atualmente em reprodução. Por exemplo: ?skip

?stop

Use este comando para parar a reprodução atual e limpar a fila de reprodução. Por exemplo: ?stop
